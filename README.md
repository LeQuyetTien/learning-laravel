1. Thiết lập môi trường phát triển cơ bản

1.1. Cài đặt laravel bằng composer
$ composer global require laravel/installer

1.2. Khởi tạo project
$ laravel new laravel

1.3. Chạy thử project
$ cd laravel
$ php artisan serve
