# laravel
Tìm hiểu Laravel

## Bài 1: Thiết lập môi trường phát triển cơ bản

1. Cài đặt laravel bằng composer
```
composer global require laravel/installer`
```

2. Khởi tạo project
```
laravel new laravel
```

3. Chạy thử project
```
cd laravel
php artisan serve
```
